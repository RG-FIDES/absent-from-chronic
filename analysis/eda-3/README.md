# `analysis/eda-3`

Scaffold for EDA-3 covering research questions **4-1** and **4-2**.

## Files

- `eda-3.R`: analytical script scaffold (development layer)
- `eda-3.qmd`: report scaffold (publication layer)
- `report-contract.prompt.md`: question and scope contract for this EDA
- `local-functions.R`: analysis-local helper function file
- `data-local/`: intermediate local artifacts
- `prints/`: exported plots
- `figure-png-iso/`: Quarto-rendered figure output

## Run

Use the EDA-3 pipeline script:

- `scripts/ps1/run-eda-3.ps1`
